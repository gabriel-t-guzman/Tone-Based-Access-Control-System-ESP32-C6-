# Tone-Based-Access-Control-System-ESP32-C6-
A real-time acoustic tone-based access control system using an ESP32-C6 and FreeRTOS. Captures audio via an analog MEMS microphone, processes target frequencies using custom 127-point FIR bandpass filters, and validates 4-digit code sequences with a noise-immune hybrid Finite State Machine. Developed at Universidade de Aveiro (DETI)
